## 导数与微分

### 概念、定理、公式

#### 导数的概念
$f'(x_0)= \lim\limits_{\Delta x \to 0}\dfrac{f(x_0+\Delta x)-f(x_0)}{\Delta x}\xrightarrow{x=x_0+\Delta x} \lim\limits_{x \to x_0} \dfrac{f(x)-f(x_0)}{x-x_0}$

- 一静 $f(x_0)$ 一动 $f(x_0 +\Delta x)$
  - 必须要有 $f(x_0)$，其中 $f(x)$ 是 $x_0$ `附近`的 $x$ 处的函数值

- 左右有别
  - $f’(x_0) \ \exists \iff f’_-(x_0) = f’_+(x_0) $
  
- 广义化
  - $f'(x_0)= \lim\limits_{\text{狗}\to 0}\dfrac{f(x_0+\text{狗})-f(x_0)}{\text{狗}}$

- ⚠️注意区分 $\lim\limits_{x \to x_0}f’(x_0)$ 与 $\lim\limits_{x \to x_0}\dfrac{f(x)-f(x_0)}{x-x_0}$ 
  - 前者为导数在该点的极限，后者为该点的导数


#### 导数的几何意义
$k = f'(x_0)$

#### 微分的概念
$\exists A \text{ not associate with } \Delta x \quad \Delta y = A \Delta x + o(\Delta x)$
- 微分
  - $\mathrm{d}y|_{x=x_0}=A\Delta x = A\mathrm{d}x$
  
#### 可微的判别
1. 写增量 $\Delta y = f(x_0+\Delta x) - f(x_0)$

2. 写线性增量（线性主部）$A \Delta x = f’(x_0)\Delta x$

3. 作极限 $\lim\limits_{\Delta x \to 0}\dfrac{\Delta y - A \Delta x}{\Delta x}$
  - 若该极限等于0，则 $y=f(x)$ 在 $x_0$ 处可微，否则不可微分

- 用简单的线性增量代替复杂的增量，且产生的误差可以忽略不计

- 与可导函数互为充要条件
  - 只对一元情况成立

#### 可微的几何解释
若 $f(x)$ 在 $x_0$ 处可微，则在该点可用切线（直线）近似代替曲线

## 导数的应用

### 概念、定理、公式

#### 几何应用

##### 极值与最值
- 与`最值`既相区别又相联系
  - 极值需要`邻域`
  - 最值只需要定义域

- 间断点可以是极值点

- 判别 
  - 必要条件（极值）
    - （可导）导数值为零
  
  - 充分条件（极值）
    - （连续、去心领域可导）导数值在该点两侧变号
    - （二阶可导）二阶导数不为零
    - （偶数阶可导、其余小阶为零）偶数阶导数不为零

##### 凹凸性与拐点
- 区间内任意两点连线的中点值大于曲线在此点的函数值为凹

- 区间内任意两点连线的中点值小于曲线在此点的函数值为凸

- 拐点：凹凸分界点

- 另一种定义
  
  $f(x)$ 在 $(a,b)$ 内连续，对 $(a,b)$ 内任意两点 $x_1,x_2, \lambda \in (0,1)$，有
  
  $f [ \lambda x _ { 1 } + ( 1 - \lambda ) x _ { 2 } ] \geq \lambda f ( x _ { 1 } ) + ( 1 - \lambda ) f ( x _ { 2 } )$
  
  则称其图形是凸的，若以上不等号方向相反，则称其图形为凹的
  
- 判别
  - 必要条件（拐点）
      - （二阶可导）则二阶导数值为零
  
  - 充分条件（凹凸性）
      - （二阶可导）
          - 大于零-凹
          
          - 小于零-凸
    
  - 充分条件（拐点）
      - （连续、去心领域二阶可导）二阶导数在该点左右领域变号
      
      - （三阶可导）二阶为零，三阶不为零
      
      - （奇数阶可导、*除一阶外*其余小阶为零）奇数阶导数不为零
